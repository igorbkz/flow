Chatbot

Este projeto oferece uma interface interativa para interação com um chatbot baseado em IA. O chatbot responde às mensagens dos usuários em tempo real, processando solicitações através de um modelo de linguagem avançado. O histórico de conversas é mantido localmente no navegador, permitindo que os usuários reiniciem o chat ou continuem conversas anteriores. A interface é projetada para ser acessível e responsiva, adaptando-se a diferentes dispositivos e fornecendo feedback visual claro durante a interação. Note que este é um produto de pesquisa em desenvolvimento e pode cometer erros.

GenarTV

Este projeto, combina inteligência artificial com arte visual em um ambiente interativo. O sistema gera imagens únicas com base em temas fornecidos pelo usuário. As imagens são exibidas dinamicamente em uma grade, onde os usuários podem explorá-las e ampliá-las através de um lightbox integrado. Além disso, oferece funcionalidades como adição de temas personalizados, gestão de histórico e pausa na geração de imagens. O projeto visa proporcionar uma experiência de descoberta artística assistida por IA de forma acessível e envolvente.
